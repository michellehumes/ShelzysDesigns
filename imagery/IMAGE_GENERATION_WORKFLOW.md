## üõ†Ô∏è Tool-Specific Tips

### For MidJourney:
1. Copy each prompt exactly
2. Paste into MidJourney Discord
3. Wait for generation
4. Use `U1`, `U2`, etc. to upscale your favorite
5. Use `/remix` to quickly change colors/names
6. Download at highest resolution

### For DALL-E 3:
1. Remove `--ar`, `--q`, `--v` parameters (MidJourney-specific)
2. Keep all descriptive text
3. DALL-E 3 is better at rendering text (names on bottles)
4. Generate through ChatGPT or API

### For Leonardo AI:
1. Use "PhotoReal" mode
2. Add "8K resolution, photorealistic" to prompts
3. Remove MidJourney parameters
