One-click content importer for ShelzyPerkins.com. Imports all pages, blog posts, menus, and settings automatically.

== Description ==

This plugin makes deploying ShelzyPerkins.com incredibly easy!

**What it imports:**
* 4 Essential Pages (About, Contact, Privacy Policy, Affiliate Disclosure)
* 6 Blog Posts with Amazon affiliate links
* Categories and tags
* Navigation menus (primary and footer)
* WordPress settings (permalinks, site tagline, etc.)

**How to use:**
1. Upload and activate the ShelzyPerkins theme first
2. Install and activate this plugin
3. Go to "Import Content" in admin menu
4. Click "Import Now" button
5. Wait 10-20 seconds
6. Done! Deactivate and delete the plugin

**Features:**
* Safe to run multiple times (won't create duplicates)
* Doesn't delete existing content
* Automatic menu creation
* Sets proper permalinks
* Configures recommended settings

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/shelzyperkins-auto-import/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the "Import Content" menu in the admin sidebar
4. Click "Import Now"

== Frequently Asked Questions ==

= Will this delete my existing content? =

No! This plugin only adds new content. It won't delete or modify existing pages or posts.

= Can I run the import multiple times? =

Yes! The plugin checks for existing content and won't create duplicates.

= What if I already have some of the pages? =

The plugin will skip any pages or posts that already exist with the same title.

= Do I need to keep this plugin activated? =

No. Once import is complete, you can deactivate and delete this plugin. It's only needed for initial setup.
