# Canva Pin Templates Guide

## Overview
Step-by-step instructions for creating on-brand Pinterest pins using Canva. All templates use Shelzy's Designs brand colors and fonts.

---

## Brand Guidelines for Pinterest

### Colors
```
Primary Pink: #FFE4E6
Gold Accent: #D4AF37
White: #FFFFFF
Black: #000000
Light Gray: #F5F5F5
```

### Fonts
- **Headlines:** Playfair Display (elegant, wedding-appropriate)
- **Body Text:** Montserrat or Open Sans (clean, readable)
- **Accent Text:** Script font sparingly for "personalized" feel

### Logo Placement
- Bottom right corner
- White or gold version on dark backgrounds
- Keep logo small but visible

## Photography Tips for Pins

### Product Shots
- Natural lighting (near window)
- Clean, uncluttered background
- Show personalization clearly
- Multiple angles

### Lifestyle Shots
- Product in use (gym, desk, wedding)
- Lifestyle props that tell a story
- Model hands for scale
- Soft, aspirational aesthetic

### Flat Lays
- Arrange products with related items
- Use consistent background
- Leave space for text overlay
- 45-degree or overhead angle

## Pin Checklist

Before publishing each pin:

- [ ] Image is 1000px wide minimum
- [ ] Aspect ratio is 2:3 (1000x1500) or longer
- [ ] Text is readable on mobile
- [ ] Brand colors used consistently
- [ ] Logo is visible but not overwhelming
- [ ] Product/CTA is clear
- [ ] Description includes keywords
- [ ] Link goes to correct product/collection
- [ ] 3-5 relevant hashtags included
- [ ] Alt text added for accessibility
