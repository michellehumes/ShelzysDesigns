## Pin Best Practices

### Optimal Pin Dimensions
- **Standard Pin:** 1000 x 1500 px (2:3 ratio)
- **Long Pin:** 1000 x 2100 px (for step-by-step tutorials)
- **Square Pin:** 1000 x 1000 px (for carousels)

### Hashtag Strategy
- Use 3-5 highly relevant hashtags
- Mix popular hashtags with niche ones
- Include product-specific + occasion-specific tags
- Research trending hashtags seasonally

### Posting Schedule
| Day | Content Type |
|-----|--------------|
| Monday | Product Pin |
| Wednesday | Educational/How-To |
| Friday | Lifestyle/Seasonal |
| Sunday | Wedding/Bridal |

### Description Formula
1. **Hook** - Grab attention immediately
2. **Benefit** - What's in it for them
3. **Differentiator** - Why yours is better (sublimation!)
4. **CTA** - What to do next
5. **Hashtags** - 3-5 relevant tags
