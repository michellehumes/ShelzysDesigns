## Usage Instructions

### Adding to Shopify

1. Go to **Shopify Admin ‚Üí Products ‚Üí Collections**
2. Select the collection to edit
3. Scroll to **Description** field
4. Click **Show HTML** (</> icon)
5. Paste the HTML description
6. Save

### SEO Setup

1. Scroll to **Search engine listing preview**
2. Click **Edit website SEO**
3. Enter the **Title** and **Meta Description** from above
4. Save

### Tips

- Keep descriptions updated with current product availability
- Update seasonal dates (like "Order by December 15")
- Add collection-specific images as hero banners
- Consider A/B testing different descriptions for conversion
