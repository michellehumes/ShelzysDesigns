# Pinterest Board Structure & Setup Guide

## Account Setup Checklist

### Step 1: Convert to Business Account
1. Go to [Pinterest Business](https://business.pinterest.com)
2. Click "Join as a business" or convert existing account
3. Fill in business details:
   - Business name: Shelzy's Designs
   - Website: [Your blog URL]
   - Business type: Blogger/Creator

### Step 2: Complete Profile
- **Profile Photo:** Logo or branded headshot
- **Display Name:** Shelzy's Designs | [Niche] (e.g., "Shelzy's Designs | Wedding & Lifestyle")
- **Bio:** (160 characters max)
  ```
  Curating the best Amazon finds for brides, preppy style lovers & organized homes. Personalized gifts, wedding essentials & aesthetic must-haves.
  ```
- **Website:** Your blog URL (claim your website for rich pins)

### Step 3: Enable Rich Pins
1. Go to Pinterest Settings > Claimed Accounts
2. Add your website URL
3. Verify using HTML tag, file upload, or DNS record
4. Apply for Rich Pins at [Pinterest Rich Pin Validator](https://developers.pinterest.com/tools/url-debugger/)

## Board Optimization Checklist

### For Each Board:
- [ ] Keyword-rich name (50 characters max)
- [ ] Compelling description (2-3 sentences, 500 characters max)
- [ ] Include 3-5 keywords naturally
- [ ] Add relevant hashtags (optional, 2-3 max)
- [ ] Set attractive board cover
- [ ] Categorize under appropriate Pinterest category

### Board Description Template
```
[What the board contains] for [target audience]. Discover [specific items 1], [specific items 2], and [specific items 3]. [Benefit statement]. #keyword1 #keyword2
```

**Example:**
```
The best bridesmaid proposal gifts and ideas for brides planning their wedding party. Discover personalized robes, will you be my bridesmaid boxes, and custom jewelry. Find the perfect way to ask your besties. #bridesmaidgifts #weddingplanning
```

## Board Cover Strategy

### Creating Consistent Covers
1. Use Canva (free) to create board covers
2. Size: 600 x 600 px (square format)
3. Include:
   - Board name in elegant typography
   - On-brand colors (blush, sage, gold)
   - Simple background or subtle pattern
   - Optional: small icon or graphic element

### Brand Color Palette for Covers
| Color Name | Hex Code | Use For |
|------------|----------|---------|
| Soft Black | #2C2C2C | Text |
| Brushed Beige | #F5F1EB | Background |
| Sage Green | #A8B5A0 | Accents (lifestyle) |
| Blush Apricot | #E8C4B8 | Accents (bridal) |
| Dusty Seagrass | #B8C5B9 | Accents (home) |

## Group Boards Strategy (Optional)

### Finding Group Boards
1. Search Pinterest for "[your niche] group board"
2. Use [PinGroupie](https://pingroupie.com) to find active boards
3. Look at competitors' group board memberships

### Joining Group Boards
- Read board rules carefully
- Follow the board owner
- Send a join request via the method specified (usually email or Pinterest message)
- Use the email template in `/resources/GROUP_BOARD_REQUEST.md`

### Group Board Best Practices
- Follow all board rules exactly
- Pin consistently but don't spam
- Engage with other pinners' content
- Share only your highest-quality pins

## Analytics to Track

### Board-Level Metrics
- **Impressions:** How often pins are seen
- **Engagements:** Saves + clicks
- **Top Pins:** Which pins perform best
- **Audience:** Who's engaging

### Review Schedule
- **Weekly:** Check top performing boards
- **Monthly:** Review all board analytics
- **Quarterly:** Adjust strategy based on data

### What to Do with Data
- **High impressions, low clicks:** Improve pin design/CTAs
- **Low impressions:** Add more pins, optimize descriptions
- **High engagement board:** Create more content for that topic
- **Low engagement board:** Consider archiving or refreshing
