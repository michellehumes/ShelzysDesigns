# Pinterest Pin Descriptions & SEO Copy

## Pin Description Anatomy

### Maximum Length
- **500 characters** (Pinterest limit)
- **Optimal:** 200-300 characters for readability

### Structure
```
[Hook sentence] [What they'll find] [Benefit/value] [CTA] [Hashtags]
```

### Hashtag Rules
- Use 3-5 hashtags maximum
- Place at end of description
- Mix popular and niche tags
- Don't use banned/spam hashtags

## Pin Title Best Practices

### Title Formulas
1. **[Number] + [Adjective] + [Product] + [Audience/Use]**
   - "10 Best Bridesmaid Gifts for Your Bridal Party"

2. **[How To] + [Action] + [Benefit]**
   - "How to Create the Perfect Proposal Box"

3. **[Year] + [Category] + [Guide Type]**
   - "2024 Wedding Day Essentials Guide"

4. **[Trigger Word] + [Product Category]**
   - "Must-Have Amazon Organization Finds"

### Title Length
- **Optimal:** 40-60 characters
- **Maximum visible:** 100 characters (truncated after)

### Power Words to Include
- Best
- Ultimate
- Must-Have
- Essential
- Perfect
- Top
- Viral
- Favorite
- Curated
- Guide

---

## Description Writing Tips

### Do:
- Start with a hook or question
- Include primary keyword in first sentence
- Add value (what they'll learn/find)
- Include clear call-to-action
- Use appropriate hashtags
- Match tone to your brand

### Don't:
- Stuff keywords unnaturally
- Use all caps excessively
- Include broken links
- Forget hashtags
- Write generic descriptions
- Exceed 500 characters
