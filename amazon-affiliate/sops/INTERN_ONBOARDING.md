## Week 2: Hands-On Training

### Day 1-2: Product Database Training

**Learn:**
- [ ] How to navigate the Master Spreadsheet
- [ ] How to find products on Amazon
- [ ] How to generate affiliate links (SiteStripe)
- [ ] How to fill in all spreadsheet columns
- [ ] How to update product status

**Practice Task:**
Research and add 5 new products to the database. Have manager review before adding more.

### Day 3: Blog Platform Training

**Learn:**
- [ ] How to log into blog platform
- [ ] How to navigate the dashboard
- [ ] How to create a new post
- [ ] How to format posts (headings, lists, images)
- [ ] How to add affiliate links
- [ ] How to schedule posts

**Practice Task:**
Create a test post (don't publish) using the template. Have manager review.

### Day 4-5: Pinterest Training

**Learn:**
- [ ] How to navigate Pinterest Business
- [ ] How to create a pin
- [ ] How to schedule pins
- [ ] How to use Tailwind (if applicable)
- [ ] How to write pin descriptions
- [ ] How to assign pins to boards

**Practice Task:**
Create 3 test pins in Canva. Schedule to a test board. Have manager review.

---

## Week 3: Canva & Design Training

### Day 1-2: Canva Basics

**Learn:**
- [ ] How to access team templates
- [ ] How to duplicate and edit templates
- [ ] How to add/replace images
- [ ] How to edit text
- [ ] How to maintain brand consistency
- [ ] How to export properly (PNG, correct size)

**Practice Task:**
Create 5 pins using existing templates. Focus on brand consistency.

### Day 3-4: Advanced Design

**Learn:**
- [ ] How to create product collages
- [ ] How to source images properly
- [ ] How to use brand colors correctly
- [ ] How to create variations of same content
- [ ] File naming conventions

**Practice Task:**
Create 3-5 pins for an existing blog post. Multiple designs, same content.

### Day 5: Quality Review

**Manager Review:**
- [ ] Review all practice pins
- [ ] Provide design feedback
- [ ] Discuss improvements
- [ ] Approve for live pinning (or more practice)

## Communication Guidelines

### Daily Communication
- **Tool:** Slack/Email/Asana (specify)
- **Updates:** Quick daily check-in
- **Questions:** Ask as they come up
- **Blockers:** Flag immediately

### Weekly Communication
- **Weekly Check-in:** [Day/Time]
- **Report Due:** End of week
- **Format:** Weekly summary (see template)

### How to Ask Questions
1. Check documentation first
2. Search existing resources
3. Write out the specific question
4. Include context/screenshots if helpful
5. Tag manager in communication tool

### How to Flag Issues
**Urgent (same day):**
- Account access issues
- Website down
- Scheduled content not posting

**Important (within 24 hours):**
- Broken links discovered
- Product availability issues
- Analytics anomalies

**FYI (next check-in):**
- Ideas for improvement
- Trends noticed
- Minor issues

## Common Mistakes to Avoid

### Product Database
- Not verifying affiliate links work
- Missing required fields
- Not updating product status
- Adding unavailable products

### Blog Posts
- Forgetting affiliate disclosure
- Broken affiliate links
- Poor image quality
- Not following SEO checklist

### Pinterest
- Pinning to wrong boards
- Missing/poor descriptions
- Inconsistent scheduling
- Ignoring engagement

### Design
- Off-brand colors/fonts
- Low-quality images
- Text too small to read
- Forgetting watermark/handle
