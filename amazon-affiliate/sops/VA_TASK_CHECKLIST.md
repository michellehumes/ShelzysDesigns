# VA/Intern Task Checklists

## Quick Reference Checklists

Print these or keep them open while working to ensure consistency.

---

## Checklist 1: Adding Products to Database

**For each new product:**

- [ ] Product is available on Amazon
- [ ] Product has 4+ stars rating
- [ ] Product has 500+ reviews (preferred)
- [ ] Product is Prime eligible (preferred)
- [ ] Product fits one of our 15 categories

**In the spreadsheet, fill in:**

- [ ] Product Name (exact Amazon title)
- [ ] Category (from approved list)
- [ ] Amazon URL (direct product link)
- [ ] Affiliate Link (generated via SiteStripe)
- [ ] Price Range ($X-$X format)
- [ ] Key Features (3-5 bullet points, pipe-separated)
- [ ] SEO Keywords (5-8 keywords, comma-separated)
- [ ] Competitors (2-3 alternatives)
- [ ] Photo Ideas (content concepts)
- [ ] Best Fit For (target audience)
- [ ] Status (Not Featured)
- [ ] Date Added (today's date)

**Final Checks:**

- [ ] Affiliate link opens correctly
- [ ] No duplicate products in database
- [ ] All required fields completed
- [ ] Spelling/grammar checked

## Checklist 2: Creating Pinterest Pins

**Before designing:**

- [ ] Know which blog post/products this is for
- [ ] Have all product images downloaded
- [ ] Review existing pins for this topic (avoid duplicates)

**Design checklist:**

- [ ] Using correct template (1000x1500px)
- [ ] Brand colors used correctly
- [ ] Font is Poppins (headlines) or Inter (body)
- [ ] Headline is clear and readable
- [ ] Text is large enough (min 24pt)
- [ ] Images are high quality
- [ ] Handle/watermark included (@shelzysdesigns)
- [ ] CTA is clear ("Shop Now," "Read More," etc.)

**Export checklist:**

- [ ] File format: PNG
- [ ] File size: Under 2MB
- [ ] Filename follows convention: `category-post-slug-variation-date.png`

**Per blog post, create:**

- [ ] Pin 1: Product collage, full headline
- [ ] Pin 2: Different layout, shortened headline
- [ ] Pin 3: Single product feature
- [ ] Pin 4: (Optional) List style
- [ ] Pin 5: (Optional) Quote/testimonial style

## Checklist 3: Scheduling Pinterest Pins

**Before uploading:**

- [ ] Pin image is ready and properly named
- [ ] Destination URL is correct (blog post or Amazon link)
- [ ] Board selection decided

**Upload process:**

- [ ] Upload pin image to Tailwind/Pinterest
- [ ] Add compelling title (50-100 characters)
- [ ] Write SEO description (use templates)
- [ ] Add 3-5 relevant hashtags
- [ ] Assign to primary board
- [ ] Schedule for optimal time slot

**For Interval Pinning:**

- [ ] Schedule to second board (7+ days later)
- [ ] Schedule to third board (14+ days later)
- [ ] Use slightly different description if desired

**Final verification:**

- [ ] Pin preview looks correct
- [ ] Link works and goes to right page
- [ ] Scheduled for future (not immediate)
- [ ] No duplicate pins same day/board

## Checklist 4: Blog Post Formatting

**Post setup:**

- [ ] Title includes primary keyword
- [ ] URL slug is clean and keyword-focused
- [ ] Meta description written (150-160 chars)
- [ ] Featured image set
- [ ] Category assigned
- [ ] Tags added

**Content formatting:**

- [ ] Introduction explains post value
- [ ] Table of contents/quick navigation added
- [ ] Each product has:
  - [ ] Product name (H3 heading)
  - [ ] Price range
  - [ ] "Shop on Amazon" link
  - [ ] Product image with alt text
  - [ ] "What You Get" bullet list
  - [ ] "Why We Love It" paragraph
  - [ ] "Best For" audience
  - [ ] Final CTA link

**SEO checklist:**

- [ ] Primary keyword in first 100 words
- [ ] Keyword in at least one H2
- [ ] All images have alt text
- [ ] Internal links added (2-3)
- [ ] External link to authority source (1)

**Final elements:**

- [ ] "Final Thoughts" section
- [ ] Related posts linked
- [ ] Affiliate disclosure included
- [ ] Pinterest image embedded with "Pin This" CTA
- [ ] Proofread for spelling/grammar

## Checklist 5: Daily Engagement

**Morning (5 min):**

- [ ] Open Pinterest
- [ ] Check notifications
- [ ] Respond to any comments
- [ ] Note any new followers

**Engagement session (10 min):**

- [ ] Search 2-3 relevant keywords
- [ ] Save 3-5 quality pins from others
- [ ] Leave 2-3 genuine comments on relevant pins
- [ ] Follow 1-2 relevant accounts (optional)

**Example engagement comments:**

- "Love these picks! The [specific item] is so cute."
- "This is so helpful for [occasion]! Saving for later."
- "That [item] is on my wishlist now!"
- Avoid: "Nice pin!" "Great!" "Check out my pins!"

## Checklist 7: Product Database Maintenance

**Weekly spot-check (10 products):**

For each product checked:

- [ ] Visit Amazon product page
- [ ] Verify still available
- [ ] Check price is accurate
- [ ] Verify affiliate link works
- [ ] Update "Last Updated" date
- [ ] Update status if needed

**Status updates:**

| Old Status | New Status | When |
|------------|------------|------|
| Not Featured | Blog Draft | When writing about it |
| Blog Draft | Published | When post goes live |
| Published | Fully Promoted | When pins created |
| Any | Needs Update | Price change >$5 |
| Any | Out of Stock | Temporarily unavailable |
| Any | Discontinued | Permanently unavailable |

**If product is unavailable:**

- [ ] Mark as "Out of Stock" or "Discontinued"
- [ ] Find replacement product
- [ ] Update any blog posts featuring it
- [ ] Update or remove related pins

## Quick Reference: File Naming

### Pin Files
```
[category]-[post-slug]-[type]-[version]-[year].png

Examples:
- bridesmaid-proposal-gifts-collage-v1-2024.png
- wedding-emergency-kit-single-v2-2024.png
- home-organizers-list-v1-2024.png
```

### Image Files
```
[product-name]-[angle/type].jpg

Examples:
- personalized-robe-front.jpg
- bridesmaid-box-contents.jpg
- tumbler-set-lifestyle.jpg
```
