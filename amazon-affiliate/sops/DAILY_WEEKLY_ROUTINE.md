# Daily & Weekly Routine SOP

## Overview
This standard operating procedure outlines the daily and weekly tasks required to maintain and grow the Amazon affiliate passive income system.

---

## Daily Tasks (15-20 minutes)

### Morning Check (5 minutes)
**Time:** First thing in the morning

- [ ] Check Pinterest notifications
- [ ] Respond to any comments on pins
- [ ] Verify scheduled pins went live
- [ ] Quick glance at yesterday's analytics

### Engagement Block (10 minutes)
**Time:** Midday or evening

- [ ] Save 3-5 pins from others in your niche
- [ ] Comment on 2-3 relevant pins (genuine engagement)
- [ ] Check for any Amazon Associate alerts/emails
- [ ] Respond to any blog comments

### Quick Monitoring (5 minutes)
**Time:** End of day

- [ ] Check scheduled content for tomorrow
- [ ] Note any viral pins or unexpected traffic
- [ ] Flag any issues for weekly review

---

## Weekly Tasks

### Monday: Planning & Review

**Time Required:** 30 minutes

**Tasks:**
1. **Weekly Analytics Review (15 min)**
   - [ ] Log into Pinterest Analytics
   - [ ] Note top 5 performing pins from last week
   - [ ] Check impressions, saves, and clicks trends
   - [ ] Identify any underperforming content
   - [ ] Log key metrics in tracking spreadsheet

2. **Plan the Week (15 min)**
   - [ ] Review content calendar for upcoming posts
   - [ ] Identify any seasonal opportunities
   - [ ] Set weekly goals (# pins, # posts, etc.)
   - [ ] Prepare list of products to feature

### Tuesday: Product Research

**Time Required:** 45 minutes

**Tasks:**
1. **Amazon Research (30 min)**
   - [ ] Check 5 products in your database for availability
   - [ ] Update any price changes
   - [ ] Research 3-5 new products to add
   - [ ] Check Amazon Best Sellers for trending items
   - [ ] Look for new seasonal products

2. **Database Updates (15 min)**
   - [ ] Add new products to Master Spreadsheet
   - [ ] Update status on products used in content
   - [ ] Mark any discontinued products
   - [ ] Generate affiliate links for new products

### Wednesday: Content Writing Day

**Time Required:** 2-3 hours

**Tasks:**
1. **Blog Post Writing (2-2.5 hours)**
   - [ ] Write one blog post draft OR
   - [ ] Edit and finalize one blog post
   - [ ] Add all affiliate links
   - [ ] Optimize for SEO (use checklist)
   - [ ] Add images

2. **Scheduling (30 min)**
   - [ ] Schedule post for publication
   - [ ] Set up any email notifications
   - [ ] Add post to internal linking plan

### Thursday: Pinterest Creation Day

**Time Required:** 2 hours

**Tasks:**
1. **Pin Design (1.5 hours)**
   - [ ] Create 5-10 new pin designs in Canva
   - [ ] Use varied templates for same content
   - [ ] Download all pins with proper naming
   - [ ] Organize in folders by category

2. **Pin Upload & Schedule (30 min)**
   - [ ] Upload pins to Tailwind/Pinterest
   - [ ] Write optimized descriptions
   - [ ] Assign to appropriate boards
   - [ ] Schedule for next 7 days

### Friday: Batch Scheduling & Optimization

**Time Required:** 1 hour

**Tasks:**
1. **Final Scheduling (30 min)**
   - [ ] Review all scheduled pins for next week
   - [ ] Fill any gaps in schedule
   - [ ] Add repins from quality sources
   - [ ] Verify all links are working

2. **Optimization (30 min)**
   - [ ] Review this week's top performers
   - [ ] Create similar pins for winners
   - [ ] Update underperforming pins with new descriptions
   - [ ] Test new pin designs/formats

### Saturday: Light Engagement

**Time Required:** 15 minutes

**Tasks:**
- [ ] Light engagement (saves, comments)
- [ ] Check for any urgent issues
- [ ] Respond to comments/messages

### Sunday: Rest + Quick Planning

**Time Required:** 15 minutes

**Tasks:**
- [ ] Quick review of weekend performance
- [ ] Preview Monday's scheduled content
- [ ] Light engagement if desired
- [ ] Rest and recharge

## Red Flags to Watch For

### Immediate Action Needed:
- Pinterest account warning/suspension
- Affiliate links not working
- Blog site down
- Sudden traffic drop (50%+)
- Amazon Associate account issues

### Address Within 48 Hours:
- Scheduled pins failing to post
- Comments/messages unanswered 3+ days
- Multiple products out of stock
- Analytics not tracking properly

### Address This Week:
- Declining engagement trend
- Underperforming categories
- Outdated content needing refresh
- New competitor strategies to analyze
