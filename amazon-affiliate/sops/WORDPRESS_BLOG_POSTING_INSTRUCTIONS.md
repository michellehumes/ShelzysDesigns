## Step 1: Log Into WordPress

1. Go to **shelzyperkins.com/wp-admin** (or log in via GoDaddy)
   - If using GoDaddy: Go to GoDaddy.com → My Products → Find the WordPress site → Click "Manage" or "Admin"
2. Enter username and password
3. You'll land on the WordPress Dashboard

---

## Step 2: Create a New Blog Post

1. In the left sidebar, hover over **"Posts"**
2. Click **"Add New"**
3. You'll see the WordPress editor (Block Editor/Gutenberg)

---

## Step 3: For Each Post, Do the Following

### A. Add the Title
- Click "Add title" at the top
- Paste the post title exactly as shown

### B. Add the Content
- Click in the main content area
- Paste the entire blog content
- WordPress will automatically format headings, paragraphs, and links

### C. Set the URL Slug
- In the right sidebar, find "Permalink" or "URL Slug"
- Click to edit and paste the provided slug

### D. Add Categories
- In the right sidebar, find "Categories"
- Check the boxes for relevant categories (create new ones if needed)

### E. Add Tags
- In the right sidebar, find "Tags"
- Add the tags listed for each post (comma-separated)

### F. Set Featured Image (Optional)
- In the right sidebar, find "Featured Image"
- Click "Set featured image" and upload an image

### G. Add Meta Description (SEO)
- If you have Yoast SEO or similar plugin installed:
  - Scroll down to the SEO section
  - Paste the meta description provided

### H. Publish
- Click the blue **"Publish"** button in the top right
- Confirm by clicking "Publish" again
