# Amazon Passive Income Product Master Database Guide

## Overview
This guide explains how to set up, maintain, and use your Master Product Database for Amazon affiliate content creation.

---

## Quick Start

### Step 1: Import to Google Sheets
1. Open [Google Drive](https://drive.google.com)
2. Click "+ New" > "Google Sheets"
3. File > Import > Upload the `PRODUCT_DATABASE_TEMPLATE.csv` file
4. Select "Replace current sheet"
5. Rename to: **"Amazon Passive Income Product Master [Year]"**

### Step 2: Initial Setup
1. **Freeze the header row**: View > Freeze > 1 row
2. **Add filters**: Data > Create a filter
3. **Color code categories** (see color guide below)
4. **Create category tabs** (optional): Right-click sheet tab > Duplicate for each category

## How to Generate Amazon Affiliate Links

### Your Tracking ID
**Shelzy's Designs Tracking ID:** `shelzysdesigns-20`

Always use this tag in your affiliate links!

### Using SiteStripe (Recommended)
1. Log in to [Amazon Associates](https://affiliate-program.amazon.com)
2. Navigate to any product page while logged in
3. Look for the **SiteStripe toolbar** at the top of the page
4. Click "Text" to get a text link
5. Copy the shortened link (amzn.to format)
6. Paste into the "My Amazon Affiliate Link" column

### Manual Link Format
If you need to create links manually, use this format:
```
https://www.amazon.com/dp/[ASIN]?tag=shelzysdesigns-20
```

Replace `[ASIN]` with the product's ASIN (found in the URL or product details).

**Example:**
- Product ASIN: B0XXXXXXXX
- Affiliate Link: `https://www.amazon.com/dp/B0XXXXXXXX?tag=shelzysdesigns-20`

### Manual Method
1. Go to Amazon Associates dashboard
2. Click "Product Links" in the navigation
3. Search for your product
4. Click "Get Link"
5. Choose "Short Link" format
6. Copy and paste

## Finding Products to Add

### Amazon Best Seller Lists
- [Best Sellers](https://amazon.com/bestsellers)
- [Movers & Shakers](https://amazon.com/gp/movers-and-shakers)
- [New Releases](https://amazon.com/gp/new-releases)

### Amazon Search Strategies
1. Search your category keyword
2. Filter by: 4+ stars, Prime eligible
3. Look for products with 1,000+ reviews
4. Check "Customers also bought" sections
5. Note products with "Amazon's Choice" badge

### Competitor Research
1. Find successful Amazon affiliate blogs
2. Note which products they feature
3. Search those products on Amazon
4. Add your own affiliate links

## SEO Keyword Research

### Free Tools
- **Amazon Search Bar**: Type your keyword and note auto-suggestions
- **Google Keyword Planner**: [ads.google.com/keywordplanner](https://ads.google.com/keywordplanner)
- **Ubersuggest**: [neilpatel.com/ubersuggest](https://neilpatel.com/ubersuggest)
- **Answer The Public**: [answerthepublic.com](https://answerthepublic.com)

### Paid Tools (Optional)
- **Ahrefs**: Comprehensive keyword data
- **SEMrush**: Competitor keyword analysis
- **Keywords Everywhere**: Browser extension

### Keyword Research Process
1. Start with your product category
2. Note Amazon's "Customers Also Asked"
3. Search in Google and note "People Also Ask"
4. Use tools to find search volume
5. Prioritize keywords with 1,000+ monthly searches
6. Include long-tail variations (3-5 word phrases)

## Pro Tips

### Product Selection Criteria
- **Price Sweet Spot**: $20-$60 (high enough commission, low enough impulse buy)
- **Review Count**: 500+ reviews minimum
- **Rating**: 4+ stars
- **Prime Eligible**: Always preferred
- **Visual Appeal**: Must photograph well

### Avoiding Common Mistakes
- Don't add products without checking availability first
- Always test your affiliate links before publishing
- Update prices monthly (they change frequently)
- Remove products that go out of stock repeatedly
- Focus on evergreen products over trendy ones

### Scaling Your Database
- Aim for 150+ products across all categories
- Each category should have 10-15 products minimum
- Update 20% of products monthly with fresh finds
- Track which products drive the most clicks/sales
