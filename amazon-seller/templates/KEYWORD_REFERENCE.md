## Backend Search Terms

Copy this block directly into your backend search terms field (under 250 bytes total):

```
custom tumbler name bottle personalized gift bridesmaid proposal box bridal party favor wedding shower bachelorette bach bride tribe maid of honor mother gym workout fitness hydration reusable eco friendly women girl teen birthday graduation nurse teacher appreciation corporate swag promotional
```

### Stainless Steel Water Bottle
```
insulated tumbler custom name bottle personalized gift bridesmaid proposal bridal party favor wedding shower bachelorette bride tribe honor mother gym workout fitness hydration stainless reusable eco women teen birthday graduation nurse teacher corporate promotional keeps cold hot
```

### Rules for Backend Search Terms
- No punctuation needed
- No repeated words (Amazon indexes automatically)
- Single space between words
- Do NOT include brand names (yours or competitors)
- Do NOT repeat words from title or bullets
- Use singular form (Amazon matches plurals)

## Keyword Implementation Checklist

### Titles
- [ ] Primary keyword in first 80 characters
- [ ] Secondary keyword included
- [ ] Use case keyword included
- [ ] No keyword stuffing (reads naturally)

### Bullet Points
- [ ] Each bullet contains 1-2 keywords
- [ ] Keywords placed naturally in sentences
- [ ] Benefits paired with features

### Backend Search Terms
- [ ] All 250 bytes utilized
- [ ] No duplicates from visible listing
- [ ] Includes synonyms and variations
- [ ] Common misspellings included

### Advertising
- [ ] High-intent keywords in exact match
- [ ] Medium-intent in phrase match
- [ ] Discovery keywords in auto/broad
- [ ] Negative keywords implemented
