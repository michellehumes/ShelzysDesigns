# Amazon Advertising Strategy

With optimized listings in place, advertising becomes your growth accelerator. Amazon PPC is essential for new products to gain visibility and build organic ranking.

---

## Why Amazon PPC Matters

| Benefit | Impact |
|---------|--------|
| Immediate visibility | Appear in search results from day 1 |
| Organic ranking boost | Sales velocity improves organic placement |
| Keyword discovery | Learn what customers actually search |
| Competitive positioning | Show up alongside established sellers |
| Scalable growth | Increase budget as you prove ROI |

---

## 1. Ad Types Overview

### Sponsored Products (Start Here)
**What:** Ads appear in search results and product detail pages
**Best For:** Direct sales, keyword targeting, new products
**Minimum Budget:** $10/day

### Sponsored Brands (Requires Brand Registry)
**What:** Banner ads at top of search with logo and multiple products
**Best For:** Brand awareness, showcasing product line
**Minimum Budget:** $15/day

### Sponsored Display
**What:** Retargeting ads on and off Amazon
**Best For:** Re-engaging shoppers, competitor targeting
**Minimum Budget:** $10/day

### Recommended Rollout Order
1. **Month 1:** Sponsored Products only
2. **Month 2-3:** Add Sponsored Brands
3. **Month 3+:** Test Sponsored Display

## 2. Sponsored Products Strategy

### Phase 1: Automatic Campaigns (Weeks 1-2)

**Purpose:** Let Amazon's algorithm discover converting keywords

**Setup:**
1. Go to **Campaign Manager > Create Campaign > Sponsored Products**
2. Select "Automatic targeting"
3. Name: `[Product] - Auto - Launch`
4. Daily budget: $15-20
5. Default bid: $0.75-1.00

**Automatic Targeting Match Types:**
| Match Type | Description | Recommended Bid |
|------------|-------------|-----------------|
| Close match | Keywords closely related | $0.75 |
| Loose match | Keywords loosely related | $0.50 |
| Substitutes | Competitor products | $0.60 |
| Complements | Related products | $0.40 |

### Phase 2: Manual Campaigns (Weeks 3+)

**Purpose:** Target proven keywords with precision

**Setup:**
1. Review auto campaign search term report
2. Identify keywords with sales
3. Create manual campaign with those keywords
4. Set match types strategically

**Match Type Strategy:**
| Match Type | When to Use | Bid Modifier |
|------------|-------------|--------------|
| Exact | Proven converters, highest intent | +20% |
| Phrase | Testing variations | Base bid |
| Broad | Discovery, research | -20% |

## 5. Key Metrics & Targets

### Primary Metrics

| Metric | Definition | Target |
|--------|------------|--------|
| ACoS | Ad Cost of Sale (ad spend / ad revenue) | 25-35% launch, 15-20% mature |
| ROAS | Return on Ad Spend (revenue / ad spend) | 3-4x launch, 5-7x mature |
| CTR | Click-Through Rate | 0.3%+ |
| CVR | Conversion Rate | 10%+ |

### ACoS Target by Phase
| Phase | Target ACoS | Rationale |
|-------|-------------|-----------|
| Launch | 35-50% | Acceptable to build ranking |
| Growth | 25-35% | Optimizing for profitability |
| Mature | 15-25% | Sustainable profitability |
| Scale | <15% | Maximum efficiency |

### Break-Even ACoS Calculation
```
Break-Even ACoS = Profit Margin %

Example:
- Selling price: $24.99
- Product cost: $8.00
- Amazon fees: $7.50
- Profit before ads: $9.49
- Profit margin: 38%
- Break-Even ACoS: 38%
```

```

│   └── Manual - Phrase - Medium Intent
└── Brand Defense (Sponsored Brands)
    └── Brand Name Keywords
```

### Naming Convention
```
[Product] - [Match Type] - [Purpose] - [Date]
Examples:
- ClearBottle - Auto - Discovery - Jan25
- SteelBottle - Exact - High Intent - Jan25
- SteelBottle - Phrase - Wedding - Jan25
```

## 7. Optimization Workflow

### Daily (5 minutes)
- [ ] Check spend is within budget
- [ ] Review any ACoS alerts
- [ ] Pause severely underperforming keywords (ACoS > 100%)

### Weekly (30 minutes)
- [ ] Review search term report
- [ ] Add converting search terms as keywords
- [ ] Add non-converting terms as negative keywords
- [ ] Adjust bids based on performance
- [ ] Review budget pacing

### Bi-Weekly (1 hour)
- [ ] Download full search term report
- [ ] Analyze keyword performance by match type
- [ ] Adjust campaign budgets
- [ ] Test new keywords from research

### Monthly (2 hours)
- [ ] Full campaign audit
- [ ] Calculate true profitability (including all fees)
- [ ] Pause/archive underperforming campaigns
- [ ] Plan next month's strategy
- [ ] Adjust overall budget

## 8. Negative Keywords

Prevent wasted spend by adding negative keywords for irrelevant searches.

### Negative Keyword List (Add to All Campaigns)

```
wholesale

clearance

blank
unprinted
```

### Product-Specific Negatives
For personalized bottles, add:
```
generic
plain
no design

```

Add as negatives unless intentionally targeting:
```
stanley
yeti
hydroflask
contigo
```

## 9. Sponsored Brands Setup (Month 2+)

### Requirements
- Brand Registry enrollment
- Brand logo (400x400px minimum)
- Custom headline (50 characters)
- 3 products minimum

### Headline Examples
```
Personalized Water Bottles | Custom Name Gifts | Ships Fast
Custom Bridesmaid Gifts | Personalized Tumblers | Made with Love
Wedding Party Favors | Custom Water Bottles | Women-Owned Business
```

### Sponsored Brands Best Practices
- Feature bestselling products
- Use lifestyle imagery in creative
- Link to Amazon Brand Store (not individual products)
- Target branded + category keywords

## 10. Campaign Launch Checklist

### Before Launching
- [ ] Listing fully optimized (title, images, bullets)
- [ ] Inventory in stock (30+ units minimum)
- [ ] Price is competitive
- [ ] Budget approved and funded

### Campaign Setup
- [ ] Campaign named with convention
- [ ] Daily budget set
- [ ] Default bid set appropriately
- [ ] Negative keywords added
- [ ] Auto targeting enabled (for auto campaigns)

### Post-Launch (First Week)
- [ ] Check impressions are generating
- [ ] Verify clicks are occurring
- [ ] Monitor ACoS daily
- [ ] Adjust bids if not getting impressions
- [ ] Add obvious negative keywords
