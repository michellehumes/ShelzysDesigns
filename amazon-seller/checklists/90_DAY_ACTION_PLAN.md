# 90-Day Amazon Scaling Action Plan

This actionable checklist breaks down the Amazon scaling strategy into daily and weekly tasks across three phases.

---

## Overview

| Phase | Days | Focus | Daily Time |
|-------|------|-------|------------|
| Foundation | 1-30 | Optimize listings, launch ads | 1-2 hours |
| Acceleration | 31-60 | Build reviews, A+ content | 1 hour |
| Optimization | 61-90 | Scale winners, expand | 30 min - 1 hour |

---

## Phase 1: Foundation (Days 1-30)

**Goal:** Optimize all listings and launch initial advertising campaigns.

### Week 1: Listing Audit & Title Optimization

#### Day 1-2: Research & Planning
- [ ] Export current listing data from Seller Central
- [ ] Research competitor listings (top 5 for each keyword)
- [ ] Document current title, bullets, and images for each product
- [ ] Create keyword master list (see [Keyword Reference](../templates/KEYWORD_REFERENCE.md))

#### Day 3-4: Title Rewrites
- [ ] Rewrite Clear Plastic Bottle title
  - Old: `Shelzy's Designs Personalized Island Love Water Bottles, 25oz, White Stainless Steel or Clear BPA Free Plastic, Flip Top Straw, Customizable`
  - New: `Shelzy's Designs Personalized Water Bottle with Name - 25oz Custom Tumbler for Bridesmaid Gift, Bachelorette Party Favors - BPA Free - Wedding Bridal Shower Gift for Women`
- [ ] Rewrite Stainless Steel Bottle title
- [ ] Evaluate Blank Tumbler strategy (pivot, bundle, or discontinue)
- [ ] Submit title changes in Seller Central

#### Day 5-7: Bullet Points
- [ ] Rewrite all 5 bullet points for Clear Plastic Bottle
- [ ] Rewrite all 5 bullet points for Stainless Steel Bottle
- [ ] Update bullet points in Seller Central
- [ ] Verify changes are live on listing

### Week 2: Backend & Images

#### Day 8-10: Backend Search Terms
- [ ] Optimize backend search terms for Product 1 (use full 250 bytes)
- [ ] Optimize backend search terms for Product 2
- [ ] Save changes in Seller Central

#### Day 11-14: Product Photography
- [ ] Audit current images (what's missing?)
- [ ] Plan photography shot list:
  - [ ] Main image (white background, personalization visible)
  - [ ] Lifestyle 1 (bridesmaid scenario)
  - [ ] Lifestyle 2 (party favors arranged)
  - [ ] Infographic (features callouts)
  - [ ] Personalization examples (3-4 names/fonts)
  - [ ] Use case grid (wedding, gym, corporate)
  - [ ] Quality close-up (sublimation detail)
- [ ] Schedule/complete photo shoot
- [ ] Edit and resize images (2000x2000px)
- [ ] Upload new images to listings

### Week 3: Advertising Setup

#### Day 15-17: Account Preparation
- [ ] Verify payment method in Seller Central
- [ ] Set advertising budget limit in Campaign Manager
- [ ] Review Brand Registry status (apply if needed)
- [ ] Research current CPC bids for target keywords

#### Day 18-21: Launch Auto Campaigns
- [ ] Create Auto Campaign - Clear Plastic Bottle
  - Daily budget: $10
  - Default bid: $0.75
  - Campaign name: `ClearBottle-Auto-Discovery-[Month][Year]`
- [ ] Create Auto Campaign - Stainless Steel Bottle
  - Daily budget: $8
  - Default bid: $0.75
  - Campaign name: `SteelBottle-Auto-Discovery-[Month][Year]`
- [ ] Add initial negative keywords to both campaigns
- [ ] Verify campaigns are running and generating impressions

### Week 4: Review Foundation

#### Day 22-24: Product Inserts
- [ ] Design product insert card (see template in [Reviews Strategy](../guides/REVIEWS_STRATEGY.md))
- [ ] Order insert cards (1,000 minimum)
- [ ] Plan insert card workflow for orders

#### Day 25-28: Review Automation
- [ ] Set up "Request a Review" process
- [ ] Create spreadsheet to track review requests
- [ ] Request reviews for all orders from past 30 days
- [ ] Research Amazon Vine requirements

#### Day 29-30: Week 4 Review
- [ ] Review ad campaign performance (impressions, clicks, spend)
- [ ] Adjust bids if not getting impressions
- [ ] Document changes made in first 30 days
- [ ] Take screenshots of current metrics for comparison

## Phase 2: Acceleration (Days 31-60)

**Goal:** Build reviews, create premium content, and optimize advertising.

### Week 5: Ad Optimization Round 1

#### Day 31-33: Search Term Analysis
- [ ] Download Search Term Report (last 14 days)
- [ ] Identify keywords with conversions (add to manual campaign)
- [ ] Identify keywords with clicks but no conversions (evaluate)
- [ ] Identify irrelevant keywords (add as negatives)

#### Day 34-37: Manual Campaign Launch
- [ ] Create Manual Exact Campaign - High Intent Keywords
  - Keywords: converting terms from auto campaign
  - Bid: 20% above auto campaign performance
- [ ] Create Manual Phrase Campaign - Medium Intent
  - Keywords: promising variations
  - Bid: Base level
- [ ] Adjust auto campaign bids based on performance
- [ ] Increase daily budget to $25-30 total

### Week 6: Amazon Vine & A+ Content

#### Day 38-40: Vine Enrollment
- [ ] Verify Brand Registry is active
- [ ] Navigate to Seller Central > Brands > Vine
- [ ] Enroll Clear Plastic Bottle (5-10 units)
- [ ] Enroll Stainless Steel Bottle (5-10 units)
- [ ] Track Vine reviewer engagement

#### Day 41-44: A+ Content Creation
- [ ] Design A+ Content layout (see [Brand Building Guide](../guides/BRAND_BUILDING.md))
- [ ] Create/gather assets:
  - [ ] Brand story copy and images
  - [ ] Comparison chart data
  - [ ] Lifestyle images for modules
  - [ ] Quality/feature close-ups
- [ ] Build A+ Content in Seller Central
- [ ] Submit for review

### Week 7: Brand Store

#### Day 45-49: Store Setup
- [ ] Create Brand Store structure:
  - Home page
  - Personalized Water Bottles page
  - Wedding & Events page
  - About Us page
- [ ] Design/upload hero banners
- [ ] Add products to category pages
- [ ] Write store copy
- [ ] Submit for review

#### Day 50-51: Product Insert Implementation
- [ ] Insert cards received and quality checked
- [ ] Add insert cards to fulfillment workflow
- [ ] Include with all new orders going forward

### Week 8: Mid-Campaign Review

#### Day 52-54: Performance Analysis
- [ ] Review all campaign metrics:
  - Total spend vs. budget
  - ACoS by campaign
  - Top performing keywords
  - Worst performing keywords
- [ ] Create optimization report

#### Day 55-58: Campaign Adjustments
- [ ] Pause underperforming keywords (ACoS > 50%)
- [ ] Increase bids on converting keywords
- [ ] Add new negative keywords from search terms
- [ ] Test new keywords discovered

#### Day 59-60: Phase 2 Review
- [ ] Document all changes made
- [ ] Compare metrics to Day 1 baseline
- [ ] Review count progress (target: 10+ reviews)
- [ ] Plan Phase 3 priorities

## Phase 3: Optimization (Days 61-90)

**Goal:** Scale profitable campaigns, test improvements, plan expansion.

### Week 9: Sponsored Brands Launch

#### Day 61-63: Sponsored Brands Setup
- [ ] Verify Brand Registry active
- [ ] Create brand logo (400x400px minimum)
- [ ] Write headline options (50 characters):
  - Option 1: `Personalized Water Bottles | Custom Name Gifts`
  - Option 2: `Custom Bridesmaid Gifts | Ships Fast`
- [ ] Select 3 products for campaign
- [ ] Set landing page (Brand Store)

#### Day 64-67: Campaign Launch
- [ ] Launch Sponsored Brands - Brand + Category keywords
- [ ] Daily budget: $10-15
- [ ] Monitor first week performance
- [ ] Adjust creative if CTR < 0.3%

### Week 10: A/B Testing

#### Day 68-72: Amazon Experiments
- [ ] Access Manage Your Experiments in Seller Central
- [ ] Create A/B test for main product title
  - Variation A: Current optimized title
  - Variation B: Alternative keyword order
- [ ] Create A/B test for main image (if you have variants)
- [ ] Let tests run for minimum 2 weeks

#### Day 73-74: Price Testing
- [ ] Analyze competitor pricing changes
- [ ] Test 5-10% price increase on best performer
- [ ] Monitor conversion rate impact
- [ ] Decide on permanent pricing

### Week 11: Scale & Efficiency

#### Day 75-79: Campaign Scaling
- [ ] Identify top 5 performing keywords across all campaigns
- [ ] Increase bids on top performers by 15-20%
- [ ] Create dedicated campaigns for best keywords
- [ ] Increase daily budget on profitable campaigns
- [ ] Target overall daily budget: $40-50

#### Day 80-81: Efficiency Audit
- [ ] Calculate true profitability per product
  - Revenue - COGS - Amazon fees - Ad spend = Profit
- [ ] Identify any unprofitable keywords/campaigns
- [ ] Make cut/scale decisions

### Week 12: Review & Plan

#### Day 82-85: 90-Day Analysis
- [ ] Create comprehensive performance report:
  - Total sales (units and revenue)
  - Total ad spend and ACoS
  - Review count achieved
  - Organic ranking changes
  - Profitability analysis
- [ ] Compare to Day 1 baseline

#### Day 86-88: Expansion Planning
- [ ] Analyze which products/keywords have room to grow
- [ ] Research potential new products to add
- [ ] Plan next quarter's strategy
- [ ] Set goals for Days 91-180

#### Day 89-90: Documentation & Handoff
- [ ] Document all optimizations made
- [ ] Create SOP for ongoing maintenance
- [ ] Set up recurring calendar reminders
- [ ] Celebrate wins!
